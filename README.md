Spring Boot Demo
---------------

#### How to use
Start the application using the ```main``` method and use the following URLs.

##### HTML view of primer list
- http://localhost:8080/

##### Primer API entry point
- http://localhost:8080/primers

##### Swagger UI
- http://localhost:8080/swagger-ui.html
